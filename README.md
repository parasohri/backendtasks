 Tech Stack

Node.js

Express.js

MongoDB + Mongoose

JWT Authentication

Redis (rate limiting & caching)

Postman (API documentation)

 

 How to Run the Project
1️ Prerequisites

Make sure you have installed:

Node.js (v16+ recommended)

MongoDB (running locally or Atlas)

Redis (for caching & rate limiting)

2️Install Dependencies
npm install

 Environment Variables

Create a .env file in the root folder  

 

Note: .env file is not included in the zip for security reasons.

Start the Server
npx nodemon


or

npm start


Server will start on:

http://localhost:5000

API Documentation

A Postman Collection is included separately

Import the collection into Postman to test all APIs easily
